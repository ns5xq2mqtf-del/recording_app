##　こだわり
    ・コメント投稿機能は公開/非公開設定可能。
    　自分の気に入った作品に関して、他の人の感想をみれるほか
    　非公開設定で投稿したコメントをマイページで一覧表示できるので、
    　自分用の感想記録としても使えます。
    ・作品へのタグ付が可能。
    　気に入った作品についているタグから似た傾向の作品を探し出せます。
